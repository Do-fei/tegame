# 剧情结构（游戏化）·《重返十七岁》

> 把线性小说切成游戏的"章节 / 场景"，并标注类型：
> 〔反转〕〔恐怖CG〕〔战斗〕〔选择分支〕〔情感锚点〕〔世界观〕
> 全作采用"双重反转"叙事——这是改编的灵魂，建议让**玩家也一起被骗**。

## 叙事诡计总览（保密设计）

- **反转一（第一季末）**：玩家全程以为在拯救"挚友董晓丽"，记忆中的美好友情、并肩打架……
  真相：那些回忆全是和**高晓媛**的；"董晓丽"是凶手借附身 / 篡改记忆伪装的。玩家亲手复活了仇人。
  → 设计：前期 UI / 对白刻意以"董晓丽"署名；合照是关键道具；结局合照"换人"即引爆反转。
### 恋爱线·叙诡铁律（恋爱设计稿同步，写死）

> 详见《青春恋爱设计 v2》。以下三条凌驾恋爱所有场景：
>
> 1. **脸/身份层锁死**：反转一前所有人间态甜场景都在『被篡改记忆/往生道』通道内（高晓媛已死、灵魂态只夜间登场），玩家感知的相貌/署名/称呼一律是『董晓丽』这层皮；高晓媛真身（脸、明亮笑眼）只存在于反转后重绘版本，白天甜场景用『回忆闪回』叙述框承载。
> 2. **真名/真脸严禁前期泄底**：署名故障闪烁永不闪出『高晓媛』或任何真名，只许乱码/像素糊块/△□占位符再复位；前期 CG 与合照正常显示『董晓丽的脸』（不用留白脸/逆光藏脸，那是元层面剧透）；终极誓言『你重生我娶你；你托生我等你』仅反转一之后解锁，前期严格无真名。
> 3. **记忆校验硬证据只用龙荻私人持有物**：他折好的卷子字迹、诺基亚草稿短信、自己录的磁带声纹/歌名、MP3 标签——这些才在『篡改记忆』可解释范围内。校友录/QQ 是服务器端纯文本现实记录，董晓丽无法篡改，不得作硬证据（只作中性情绪铺垫，用字符画签名）。

- **反转二（终章前）**：以为董晓丽是元凶。真相：她被泰国巫妖**友田**附身；而友田并非护子烈女，
  是杀子鬼妻。善恶层层翻转，引向"该死的必须死"的宿命主题。

## 章节流程

### 序章 · 能看见鬼的眼睛　〔恐怖CG · 世界观铺垫〕
- 场景：深夜公司 / 保安亭「绿衣女」/ 楼道「黄衣小女孩童童」/ 回家路
- 机制教学：观察 → 对话 → 第一次"看见不该看见的东西"
- **选择**：要不要把糖给小女孩？（埋下童童善意线）
- 物业邮件揭示：女孩十天前已遇害 → 收束序章惊悚

### 第一章 · 墙上的她　〔恐怖CG · 进入往生道〕
- 场景：卧室「干尸女鬼」（铃兰中学校牌）/ 衣橱隧道 / 重返十七岁教室
- **名场面 CG**：女鬼挂衣架、后脑勺贴墙、铃兰校牌特写
- 机制教学：往生道规则（午夜、镜中无影、真实触感）

### 第二章 · 往生道的"她"　〔世界观 · 选择陷阱〕
- 场景：高二教室 / 无声课间操 / 热点餐厅
- "董晓丽"讲解 涅 · 渡者 · 往生道 · 三次法则
- **关键选择**：往生道的奶茶要不要喝？（不可进食禁忌 → Bad End 教学）
- 道具：MP3 录音（跨界留存对白）；满脸嘴的「鬼童子」来袭

### 第三章 · 大巴上的复仇　〔战斗 / 解谜 · 选择陷阱〕
- 场景：野炊时间线 / 给司机下催眠药 / 空中尾随 / 翻车
- **重大抉择**：是否执行"复仇"（玩家以为在救挚友，其实在复活凶手）
- 结算：809 病房地址 → 现世医院

### ★ 反转一 · 照片里的真相　〔大反转 · 情感锚点〕
- 触发：母亲看合照——"这是高晓媛啊，不是董晓丽"
- **母亲 NPC 逻辑（写死·方案 A『母亲也被骗』）**：生日家宴妈妈见到、当成未来儿媳的也是『董晓丽』这层皮（与玩家同被骗）。反转时母亲是看着旧照片（私人持有的现实物证，未被篡改）+ 龙荻记忆解封后才纠正『这不是董晓丽……这是高晓媛』，她与玩家同步震惊——母亲不凭空知道真名。
- 记忆恢复闪回；高晓媛灵魂归来，49 天倒计时启动
- **情感**：「你重生，我娶你；你托生，我等你」首次确立
- （＝ 原著"第一季 完"）

### 第四章 · 灭却师与狗　〔集结 · 喜剧〕
- 唐可欣夜闯（灵瞳）→ 奥兰多（童童）觉醒话痨 → 大飞身世反转 → 老金登场
- 机制：队伍系统 / 各人能力介绍；恶灵气味探测（奥兰多）
- **选择**：如何说服 / 取信唐可欣

### 第五章 · 随缘楼探灵　〔恐怖关卡 · 怨念溯源〕
- J 大随缘楼 201 / 红衣学姐传说 / 董晓丽怨念起点
- **战斗**：红衣学姐 BOSS（后被大飞契约收服）
- 揭示：现世董晓丽苏醒、巫妖入侵自身往生道

### ★ 反转二 · 鬼妻友田　〔大反转 · 恐怖高潮〕
- 泰国篇：黑法师老太太 / 友田鬼牌 / 最终战
- 真相独白：董晓丽是被附身的受害者；友田＝杀子鬼妻
- **BOSS**：友田巫妖（吸食涅能量进化）

### 终章 · 重返十七岁　〔救赎 · 续作伏笔〕
- 老金禁术「时空逆转」→ 全员回到高二、保留记忆
- 大团圆：教室、当众一吻、"媳妇儿！"
- 伏笔：会所二楼神秘女人 / 十字会 / 罗刹海市 / 老金女儿 → 第二部《猎人》

### 恋爱机制·阶段约束（恋爱设计稿同步）

- **阶段 3 切片只放 1 个恋爱名场面**：`ch2_milkshake`（共享奶昔被掐）——一句台词 + 一次心动选择 + 一个 `flag`（如 `kept_milkshake_note`）+ 一次署名渲染（董晓丽），端到端跑通双端。其余 30+ 恋爱场景进阶段 4 内容填充 backlog。
- **阶段 3 只保留『薄荷便签』一条机制**；称呼系统/可重访据点/成对 CG 解锁/亲手贴标签全部为阶段 4+ 可选增强。
- **schema 未定型前不焊死字段**：阶段 3 复用现有 `flag` 机制承载便签；结构化 `keepsake` 字段仅作阶段 3 引擎评审时的候选提案。
- **消灭隐形进度条（与反刷分一致）**：反转署名重写演出按『阈值布尔门』——≥1 张软心便签播完整版、0 张播保底旁白版，不按便签数量缩放；合照换脸为单次剧情事件；CG 解锁『到达节点即给』。强制归档便签（奶昔/玻璃瓶救场/生日家宴）保证反转重写至少有 3 张可重写的牌。
- **恋爱与结局解耦**：隐藏结局门槛仍只由 MP3/合照/童童提示决定，恋爱机制不参与门槛、不接入任何可累积变量、不影响分支可达性。
