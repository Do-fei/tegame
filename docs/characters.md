# 人物档案 ·《重返十七岁》

> 每个角色含：身份 / 性格 / 关系 / 能力 / **像素立绘要点** / 游戏定位。
> 立绘要点是给"现代彩色像素"立绘用的视觉关键词。

## 主角团

### 龙荻 — 主角 / 玩家视角
- **身份**：B 市互联网公司金融分析员；曾是铃兰高中"头号不良少年 / 少爷"
- **性格**：外痞内善，重情义，嘴硬心软，对母亲既愧疚又深爱
- **关系**：高晓媛（初恋 / 此生所求）· 大飞（小弟兼猎人战友）· 母亲（相依为命）
- **能力**：天生阴阳眼；觉醒「猎人术·赤焰」；钨钢匕首（大飞所赠，可被法力加持）
- **立绘要点**：高中＝黄发痞气、单肩书包、校服外套散扣；成年＝衬衫西裤、眼神疲惫而温和；战斗＝赤红火焰缠绕双拳
- **游戏定位**：玩家化身；选择与猎人术的执行者

### 高晓媛 — 女主 / 初恋
- **身份**：高二转学生；龙荻的潜在恋人
- **性格**：表面文静淑女、实则飒爽，会陪龙荻打架；心灵手巧（自制薄荷香水、会做饭）
- **恋爱台词配平（写死）**：『飒/有主心骨』的实证密度必须 ≥『软/脸红』。可笑可偶尔脸红，但更多是飒——能三言两语镇住挑事混混、考前反过来罩着慌神的龙荻、被占便宜立刻一句顶回且不脸红。削减『眼睛弯弯/眨眼/仰头/抿嘴笑』卖萌镜头语言，不给撒娇波浪号『~』，撒娇腔统一替换为『飒中带笑的损』（如『谁让你嘴比心实诚』）。
- **标志**：淡淡薄荷香、薄荷叶吊坠（灵魂收纳形态）
- **挂坠设定修正（写死）**：弥勒佛象牙挂坠是高晓媛**唯一**的遗物/灵魂本体，**不存在『同款』**。随缘楼定情走**单向赠予**——她摘下贴身那只塞给龙荻，龙荻回赠别的信物（打火机/校牌）。这样『贴身一季的定情物其实是真爱的尸骸/灵魂本体』在反转时才成立。

- **真相**：被董晓丽（友田附身）剪断车闸害死；灵魂存续仅 49 天
- **立绘要点**：人间＝白衬衫深灰短裙、明亮笑眼；灵魂态＝半透明蓝 / 绿光、发丝飘动；吊坠态＝薄荷叶绿色挂坠
- **游戏定位**：情感核心；夜间登场的"涅"伙伴

### 奥兰多（童童）— 吐槽役 / 向导
- **身份**：张红老师之女童童（5 岁半），在龙荻公司大楼遇害；残魂寄身于龙荻的旧玩具狗"奥兰多"
- **性格**：话痨、傲娇、贪吃、爱看言情剧、嘴损心善
- **能力**：鼻子可嗅恶灵 / 巫妖气味；继承玩具狗属性（白天也能活动）
- **立绘要点**：洗得发白的毛绒狗，后被烟头烧掉鼻子（固定笑点）；童童本体＝黄色连衣裙＋旧兔玩偶
- **游戏定位**：解说 / 提示系统拟人化；喜剧调剂；恶灵探测器

### 唐可欣 — 灭却师
- **身份**：猎人（灭却师），大飞师妹
- **性格**：冷脸毒舌、外冷内热、战斗力偏弱（团队笑点之一）
- **能力**：左眼「灵瞳」（蓝色机械感，可视善恶 / 血债，耗元气）；武器「降魔杵 / 打狗棍」
- **立绘要点**：黑色运动服、赤脚、白色医用眼罩（揭开＝蓝光波斯猫眼）、铜色短杵；身带中药味
- **游戏定位**：嘴上的战力担当 / 真·辅助

### 大飞 — 兄弟 / 银牌猎人
- **身份**：龙荻高中跟班，实为银牌猎人（阴阳眼家族）
- **性格**：仗义、贱兮兮、随叫随到、嘴欠
- **能力**：双匕首；「猎人术·玄冰咒」；猎人术寻踪 / 瞬移
- **立绘要点**：与龙荻同款匕首、嘻哈痞帅、招牌欠揍笑
- **游戏定位**：战友＋身世反转的信息揭示者

### 铁罗汉 / 金麟刚 / 老金 — 师父
- **身份**：金牌猎人，大飞与唐可欣的师父
- **性格**：嬉皮笑脸大胃王 ＋ 关键时刻一本正经的强者；自称"贫僧在清修"
- **能力**：金属扇 / 法眼 / 封印肉身；禁术「猎人术·时空逆转」
- **立绘要点**：近三百斤光头胖和尚、僧袍、青蓝挎包、佛珠；战斗＝身后金色法轮
- **游戏定位**：导师 / 力量天花板 / 喜剧担当；女儿伏笔承载者

## 反派 / 关键角色

### 董晓丽 — 班花 / 表里反转
- **身份**：高二班花、班长；家境优渥、高傲、爱搞小团体
- **真相**：泰国巫妖"友田"附身 → 被迫剪车闸害死高晓媛 → 自杀成涅 → 复活成巫妖
- **复杂性**：本性有少女嫉妒但非大恶；既是加害者也是受害者
- **立绘要点**：清秀高傲、校服；附身 / 巫妖态＝赤红猫瞳、黑雾缠手、青筋暴起、眼白消失
- **游戏定位**：第一季"伪挚友"诡计核心；第二季 BOSS

### 友田 / 灵妻友田 — 真·恶源
- **身份**：泰国传说"灵妻"，实为产后抑郁杀子的"鬼妻"巫妖
- **黑暗真相**：杀亲子（蒸锅）、杀邻居、自刎；被误传为护子烈女，黑法师取其墓土制鬼牌
- **立绘要点**：无五官白脸（塑料球感）/ 黑色长袍；进化态＝吸食涅能量膨胀、血色光
- **游戏定位**：最终 BOSS / 第二次大反转

### 马威廉 — 跟班 / 工具人凶手
- **身份**：纪律委员，董晓丽舔狗，父亲是训导主任
- **行径**：被巫妖控制，于公司大楼杀害童童
- **立绘要点**：唯唯诺诺；被附身＝垫脚尖、白眼、僵硬姿态
- **游戏定位**：恐怖事件执行者；童童线仇敌

### 张红老师 — 班主任 / 隐藏羁绊
- **身份**：龙荻高中班主任，湖北人，严厉但真心待学生；童童的母亲
- **细节**：中考备酸梅汤 / 龙虎人丹；高三忽然提前退休
- **游戏定位**：情感伏线（童童母亲身份的揭示）

### 红衣学姐 — 章节型恐怖 BOSS
- J 大随缘楼 201 的都市传说怨灵；四肢被钉墙惨死；红裙、歪头 90°、空眼眶、裂到耳根的嘴
- **游戏定位**：探灵篇恐怖关卡 BOSS；后被大飞契约收服
