# 青春恋爱设计 ·《重返十七岁》（评审修订版 v2）

> 本文档综合 5 个视角（原著甜点考据 / 新增青涩场景 / 轻量恋爱机制 / 甜恐穿插与反转武器化 / 台词分寸护栏），并按 tone / canon / design 三路评审意见定稿。
> **第一铁律**：这段恋爱既是情感核心，也是反转的弹药——玩家对"董晓丽"投入越深，结局照片"换人"时越心碎。所有甜，本质上都是和**高晓媛**的，写时心里认定是她，UI 上署名留扣子。
> 基调护栏：青涩（初恋的笨拙、心动、欲言又止），不油腻、不超龄、合年代（2000 年代初中国高中）、不破坏恐怖悬疑主线、不与已定反转冲突。

## ★ 三条总规则（评审修订后写死，凌驾全文）

> 本节是三路评审定稿的总闸，下文任何场景 / 机制与本节冲突，以本节为准。

### A. 篡改记忆的「脸 / 身份层」锁死规则（canon）
所有反转一之前的人间态甜场景，**一律发生在"被篡改记忆 / 往生道"通道内，不是现实人间态同框**——高晓媛已死、灵魂态只在夜间登场。在这些被改写的记忆里：
- 玩家感知到的**相貌、署名、称呼一律是"董晓丽"这层皮**；高晓媛的真身（脸、明亮笑眼）只存在于反转后的重绘版本。
- 叙述上用**"回忆闪回"框**承载白天密集甜场景（"……那时候你记得是这样的"），降低"她大量正常人间态同框"与"她已死"的张力；适当削减白天同框的绝对密度。
- 凡"记忆校验硬证据"只能挂在**龙荻私人持有物**（他自己折好的卷子、他录的磁带、他 MP3 里的标签）上——这些才在"篡改记忆"可解释的范围内。**校友录 / QQ 等服务器端现实记录不能被董晓丽篡改，不得作硬证据**（见 §二修订）。

### B. 真名 / 真脸严禁前期泄底规则（canon）
- **署名故障闪烁永远不得闪出"高晓媛"或任何可识别真名**。只允许出现：无意义乱码 / 像素糊块 / 名字第一笔残影 / △□等占位符，再复位成"董晓丽"，制造"这里有点不对劲"的不安，而非递出答案。**真名首次可读，严格锁死在反转一之后。**
- **前期 CG 正常显示"董晓丽的脸"**，不用"刻意逆光藏脸 / 留白脸"——那是类型化的"会换脸反转"元信号，反而软化冲击。若要留扣子，只用极隐晦的非脸部细节（薄荷绿光晕一闪、吊坠特写）。
- **终极誓言"你重生我娶你；你托生我等你"仅限反转一之后解锁**，前期机制清单不得触发；任何"喊对真名 / 誓言"的内容都在反转扳机之后才出现，前期严格无真名。

### C. 阶段切片与机制减法规则（design）
- README 阶段 3 = **"一个能在终端 + 网页跑通的名场面（对话 + 立绘 + 选择）"**。本稿**只把 1 个名场面（"共享奶昔被掐"）放进阶段 3**，其余全部进阶段 4 内容填充 backlog（见 §六分栏）。
- 阶段 3 **只保留"薄荷便签"一条机制**。称呼系统、可重访据点、成对 CG 解锁、亲手贴标签——全部降级为阶段 4+ 可选增强，明确标注"非切片范围"。
- **彻底消灭隐形进度条**：所有"多 = 更强"的递进反馈，改成"阈值 = 触发"的布尔门（详见 §三）。
- **恋爱机制与结局 / Bad End 完全解耦**：隐藏结局门槛只由主线线索（MP3 / 合照 / 童童提示）决定；恋爱不接入任何可累积变量、不影响分支可达性（详见 §三）。

  - 机制："反复刷新"=暗恋的笨拙具象化（**删去"截图存三份备份"的揭底**，tone）；**字符画薄荷叶是中性涂鸦，前期不署名/不明示绑死董晓丽，避免懂行玩家提前连线（canon）**。

## 三、轻量恋爱机制：薄荷便签（The Mint Notes）

> 一套**零数值**的恋爱机制——没有好感度条、没有刷分、**没有任何隐形进度条**。一本随身便签本，只在剧情触发或你做出"心动选择"时保存一张"心动便签"（一句台词+一个感官锚点）。机制的全部重量都压在反转上。
> **design 修订核心**：阶段 3 只实现这一条机制；所有"多=更强"改为"阈值=触发"布尔门；与结局彻底解耦。

### 三条极简规则
1. **心动便签（替代好感度）**：剧情偶现"心动选择"——两个都成立的反应（一个嘴硬痞气 A，一个露出软心 B），无对错、无加分提示。任选都推进剧情；若选 B，系统**静默**撕下一张薄荷绿小卡（场景一句话+当事人台词+感官锚点）。玩家只感到"这一刻我记住了"，看不到任何"+5 好感"。
2. **强制归档便签（保底，design 修订）**：为防止"走极痞路线一张软心便签都没撕、反转重写空播"，**几个第一季必经的原著名场面（共享奶昔、玻璃瓶救场、生日家宴）设为"强制归档便签"——剧情自动撕下，不走可选 choice**。保证无论玩家怎么选，反转一"署名重写"都有**至少 3 张可重写的牌**。可选软心便签只在此基础上叠加，而非唯一来源。
3. **署名诡计（与反转死死绑定）**：每张便签底部手写署名。第一季全程 UI 渲染成"——董晓丽"。**署名故障闪烁严守总规则 B：永不闪真名，只闪乱码/糊块/占位符。**

### 反转重写：阈值布尔门，不是进度条（design 修订）
- 〔反转一〕母亲纠正"这是高晓媛"后，便签本被强制打开，逐页播放"署名重写"动画——每张"——董晓丽"被划掉、墨迹晕开、重写成"——高晓媛"，薄荷锚点点亮成"原来一直是她"。
- **演出长度不再按便签数量缩放**（消灭隐形进度条）。改为**布尔阈值**：
  - **有 ≥1 张软心便签** → 播**完整版**重写演出（含 3 张强制归档 + 玩家额外撕下的若干）。
  - **0 张软心便签（全选 A）** → 仍有 3 张强制归档可重写 + 播一段**保底旁白版**（"那些名字，本来就不该是她的——是你忘了，我替你记着"）。
- 攻略党**无数可刷**：多撕便签不让演出"更长更疼"，只是多几张被划掉的纸；少撕也成立。

### 心动选择范例（去油腻 / 飒中带笑修订）
- 共享奶昔：A〔痞〕「这杯我买的，凭啥分你一半」/ B〔软〕「……喏，插管那头给你，别一大口，齁着活该」。选 B→便签『一杯奶昔两根管，她一大口，我假装心疼真心疼』。
- 自行车大梁下坡风大：A〔痞〕「坐稳了别叫，丢人」/ B〔软〕（不说话，悄悄拢胳膊挡风，闻到薄荷）。选 B→便签锚点=薄荷香，为反转埋雷。**（注：此为全季"拢/挡"类身体接触额度之一，受 §五量化指标约束。）**
- 课堂传纸条老师快走过来：A「把纸条吞了」/ B「反手把纸条扣她课本下，眼神错开装看黑板」。选 B→便签『纸条扣过去那下，俩人都装没事，耳朵是烫的』。**（删去"指尖在桌底碰0.5秒"的微观特写，改走"物+眼神错开"，tone。）**
- 随缘楼黑暗互赠护身符：A「嘁，谁要你的」/ B「……这个给你，别弄丢」（她把贴身挂坠塞过来，龙荻回赠打火机）。选 B→弥勒佛象牙挂坠便签（单向赠予版）。
- 她砸瓶救你喊"敢动我爷们儿"：A「谁是你爷们儿」/ B「（把战利品往她兜里塞）下回躲我后面」。选 B→并肩打架便签。

### 收集系统（同一本两副面孔）—— 阶段 4+ 增强
- **心动便签（普通收集）**：每个心动选择存一张，薄荷绿散页夹在便签本里。终端=ASCII 卡片翻阅；网页=可掀页笔记本。
- **隐藏 CG·心动定格（稀有收集）**：**CG 解锁改为"到达该剧情节点即给"，不再"集齐成对线索"（design 修订，消灭刷取动机）**。前期 CG **正常显示"董晓丽的脸"（不留白，总规则 B）**，反转后同一构图重绘为高晓媛明亮笑眼。
- **合照（剧情核心道具，非自由收集）**：全程只有一张，反转的扳机，被剧情交到你手里，呼应原著。**换脸是单次剧情事件，不随场景递进"越靠越近"（design 修订，消灭进度条）。**
- **薄荷便签·完全版（真相全知线专属）**：走隐藏结局多出一沓"被董晓丽篡改前的原始便签"，逐字对照，给"明察"线额外的痛与了然。**注意：隐藏结局门槛只由主线线索决定（见下），便签完全版是"到达隐藏线后额外的彩蛋内容"，不是门槛条件。**

### 与结局的挂钩（彻底解耦，design 修订）
1. **重写强度 = 布尔门，不改主线走向**：见上，有/无软心便签决定播完整版/保底版，**不存在"越多越疼"的可刷量**。
2. **隐藏结局门槛：恋爱机制完全不参与**。隐藏结局仍**只由 MP3 / 合照 / 童童提示这些主线线索决定**（与 script-outline 既有"集齐 MP3/合照/童童提示"一致）。**"薄荷香锚点未点破"已从门槛中移除**——玩家永远没有"为结局去刷恋爱"的动机。
3. **第三章"复仇动力"= 纯演出旁白，不接变量**：反转后旁白点题"正是这些回忆让你决定动手"，但**不接入任何可累积的"决心值"，不影响任何分支可达性**（design 修订）。Bad End 仍由禁忌/穿越次数决定。
4. **终章回响**："重返十七岁·当众一吻"时，便签本自动新增一张永不褪色的便签，署名第一次由玩家在反转后亲手写下"——高晓媛"，与开头诡计署名形成闭环。

### 反刷分（根本没有可刷的数）
- 无数值、无进度条、无加分提示、**无"多=更强"的布尔以外反馈**——玩家无从最大化。
- 便签是事件触发非数值累积，与剧情节点绑死，不能回头农场；错过也成立。
- CG 解锁是"到达节点即给"，前期脸正常显示董晓丽，无攻略式刷取动机。

### 工程可行性（阶段切片分级，design 修订）

**阶段 3（垂直切片，只做这些）：**
- **数据**：**先不焊死 keepsake 字段名**。阶段 3 复用现有 flag 机制承载便签（如 `flag: kept_milkshake_note`），choice 形态与 script-outline 既有 `{text, goto, flag}` 一致。
  ```json
  { "text": "……喏，插管那头给你", "goto": "ch2_milkshake_after",
    "flag": "kept_milkshake_note" }
  ```
- **core**：维护一个 keepsakes 列表（可直接由若干 flag 派生），进存档；`reversalDone` 标志。这些**标注为"依赖 core 存档系统就位后实现"**。
- **终端最小版**：便签 = **静态 ASCII 卡片**；署名重写 = **一行 ANSI 文字把"董晓丽"替换成"高晓媛"**（最小可行，不做逐帧动画）。
- **网页最小版**：便签 = 像素 UI 面板静态卡片；署名重写 = 文字替换 + 一次墨迹晕开（单帧即可）。

**阶段 4+ / 阶段 5 打磨（明确标注"非切片范围"）：**
- 翻页便签本（接管输入循环/分页）、墨迹晕开逐帧、薄荷绿香气粒子特效 → **网页专属增强 / 终端阶段 5 打磨项**。
- CG 双版贴图（董晓丽脸 / 反转后高晓媛笑眼），按 `reversalDone` 切换。
- 称呼系统、可重访据点、亲手贴标签 → 阶段 4+ 可选增强。
- **承认双端不必像素级对等**：终端的动画/粒子能力本就受限，不强求与网页对等，不把终端成本说成"极便宜"。

**候选 schema 提案（待阶段 3 schema 评审，不作承诺）：**
> 若阶段 3 引擎评审通过，可把 flag 升级为结构化字段。以下仅为**提案**，schema 未定型前不视为承诺：
```json
{ "text": "……喏，插管那头给你", "goto": "...",
  "keepsake": { "id": "ks_milkshake", "anchor": "mint|shared",
    "line": "一杯奶昔两根管…", "signedAs": "dongxiaoli" } }
```
> 反转时 `rewriteSignatures()` 把 `signedAs` 从 `dongxiaoli`→`gaoxiaoyuan` —— 同样标注"依赖 core 存档与全局状态迁移就位后实现"。

## 四、与反转·恐怖的融合

### 甜恐穿插原则（三明治诱饵法 / 薄荷与血）
恋爱不是恐怖之外的奖励，而是恐怖叙事的**"诱饵层"**。每段甜都在垫高情感投资；每次恐怖都在收割。两线共用同一套机制（阴气压力条、午夜法则、不可进食、MP3 录音、往生道篡改）。

1. **甜在前、恐在后，绝不反过来**：先发糖再扎刀。先让玩家在卡座共享奶昔笑出来，下一屏才让对面的"她"在玻璃倒影里没有影子。
2. **午夜法则=节奏天然分割器**：白天=准备+恋爱（阳光、薄荷香、笨拙心动），夜晚=行动+恐怖（涅现形、阴气侵蚀）。同一个"她"白天是初恋少女、午夜是恐怖涅形——玩家在"我爱的人"和"我害怕的东西"间被迫切换。
3. **阴气压力条=甜度的隐形税**：每次甜场景让玩家放松（轻微回血/降阴气），代价是屏蔽掉一两条本该警觉的恐怖线索（倒影、薄荷香变腥、署名一闪）。越沉浸恋爱越看不见破绽——"投入恋爱"="主动选择被骗"。
4. **同一道具，双重温度（凶器化分级，tone 修订）**：每个甜道具预埋成凶器，但**分级处理，避免把口腔共用物明示成禁忌后果的成人惊悚气味**：
   - **保留（不涉身体亲密）**：薄荷香水（甜→午夜变血腥/福尔马林味）、自动铅笔笔尖（往生道课桌下划血痕）、弥勒佛象牙挂坠（定情→其实是高晓媛遗物/灵魂本体，握久发凉）。
   - **改克制（原"共享奶昔吸管=不可进食禁忌 Bad End 诱饵"）**：去掉"口腔共用物=禁忌后果"的暗示，改为**午夜法则下的灵异留白**——"她那半杯空了，吸管上没有唇印"，靠"该有的痕迹消失了"制造瘆人，而非把吸管明示成 Bad End 诱饵。
   - **删去对恋爱场景反复贴"亲密/性张力"的元标注措辞**，全文改用"青涩/心动"。
5. **署名诡计贯穿到甜场景 UI 层（守总规则 B）**：前期对白署名"董晓丽"，越甜的场景署名越要"短暂闪烁/糊一下"——但**只能糊成乱码/像素块/占位符再复位，永不闪出真名**。当时以为是浪漫小 bug，反转后回看全是尸检报告。
6. **恐怖从甜的内部渗出**：最好的恐怖不是另起场景，而是让甜场景"当场腐烂"——自行车那段薄荷香里突然混进一丝腥；她在你怀里笑，但车筐倒影里坐着别人；她"你属狗啊"闻你嘴，呼吸却是凉的。甜恐共享同一帧。
7. **留白=最大的恐怖与最大的甜共用同一手法**：初恋的"欲言又止"和恐怖的"细思极恐"是同一个引擎。一句没说完的"其实我……"既是少女羞涩，也是涅想说"我已经死了"。第一遍读成甜，第二遍读成遗言。

### 反转武器化（让恋爱投入在"照片换人"时变成最大伤害）
核心机理：伤害=玩家的**"主动共谋感"**。不是"我被剧情骗了"，而是"是我亲手把甜蜜归在错误的人身上，我亲手复活了凶手"。

1. **亲手归档（阶段 4+ 增强，非切片）**：关键甜场景结束给一个看似无害的轻机制——把回忆"写进 MP3 标签/给纪念品贴标签"，系统默认填"董晓丽"，玩家可改可不改。反转时把玩家亲手贴的标签当面划掉重写。**（注：此为阶段 4+ 增强，阶段 3 不实现；且不接入任何可累积变量。）**
2. **合照作为唯一贯穿道具**：一张大头贴/合照序章就进物品栏。**前期正常显示"董晓丽的脸"（总规则 B，不留白）**。反转时这张你看了一整季的照片当面像素级重绘换脸（董晓丽→高晓媛），薄荷叶吊坠浮现。换脸是**单次剧情事件**，不随场景递进。
3. **复用"错救凶手"分支，让甜成为复仇燃料（纯演出，design 修订）**：第三章玩家以为为挚友复仇（其实复活凶手）。反转后**旁白点题**"正是这些回忆让你决定动手"——但**不接入任何可累积"决心值"、不影响分支可达性**。爱=罪，靠叙事而非数值传达。
4. **双声道闪回——同一段甜，两个女孩**：反转记忆恢复演出，把每个标志性甜场景原地重播，立绘/声线/署名从董晓丽切成高晓媛，**台词一字不改**。"原来一直是她，是我把她忘了"。
5. **薄荷香的回收**：全季让玩家在甜场景反复闻到薄荷香却归因"董晓丽"。反转瞬间把吊坠这件**单向赠予的"定情信物"**揭示为高晓媛的**遗体/灵魂本体**——贴身携带一季的定情物其实是真爱的尸骸。最甜的信物=最恐怖的真相。
6. **倒计时立刻接管，不给缓冲**：照片换人=第一季终，紧接"高晓媛灵魂仅存 49 天"倒计时与"你重生我娶你；你托生我等你"誓言（**首次出现即此处，总规则 B**）。把愧疚（反转）和紧迫（倒计时）焊死。

### 母亲 NPC 逻辑（canon 修订，写死）
> 原稿矛盾：母亲既在甜蜜期"把被冒名的她当未来儿媳"，又在反转时叫得出"高晓媛"真名——若母亲记忆也在篡改通道里、也署名董晓丽，她凭什么突然说出真名？
> **定稿采用方案 A「母亲也被骗」**：
> - 生日家宴，妈妈见到的、当成未来儿媳的也是**"董晓丽"这层皮**（与玩家同被骗）。
> - 反转时，母亲是**看着旧照片 / 被龙荻唤醒记忆后**才纠正——"这不是董晓丽……这是高晓媛啊"，**她和玩家同步震惊**（受害者一起被骗，比"早就知情"更痛）。
> - 机制说明：母亲不凭空知道真名；真名是"旧照片这个未被篡改的现实物证 + 龙荻记忆解封"共同唤出的。这与总规则 A（硬证据须是私人持有的现实物）一致——**合照（私人持有的实体照片）是合法的真相载体**。

## 五、台词分寸护栏

### ★ 量化指标（tone 修订，硬上限）
- **全季"身体接触"类甜点（牵手 / 搂 / 拢 / 贴脸闻香）总数 ≤ 4 处**。当前定稿身体接触额度分配：①大梁拢胳膊挡风（§三范例）②随缘楼黑暗互赠挂坠的手递手（唯一的"黑暗中接触"，§五配对）③玻璃瓶救场后把战利品塞她兜 ④终章当众一吻。**晚自习停电改为非接触版，不占额度。**
- **"一吻"仅终章一次**，前期一律不写。
