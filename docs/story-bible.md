# 世界观设定集 ·《重返十七岁》

> 把原著中散落的设定，整理成游戏可直接引用的"规则系统"。
> 这些规则同时也是玩法机制的来源（穿越次数＝资源、不可进食＝禁忌、阴气＝压力条……）。

## 一、基础概念

### 阴阳眼
天生能看见鬼魂等超自然存在的特异体质。平时"未开启"，当强大的怨灵主动接近时会被逐渐激活。
- 龙荻：天生阴阳体质，被两只涅接近后觉醒
- 大飞及其父亲：阴阳眼家族
- 唐可欣：左眼为「灵瞳」（蓝色，可视善 / 恶 / 血债）

### 涅（niè）
枉死、冤死、心有不甘的冤魂。因怨念过大无法投胎，滞留人间。
- 会用自己的怨气制造一个结界空间「往生道」
- 每只涅有一次被拯救的机会：回人间 / 入冥界轮回

### 往生道
涅用怨气凝成的结界世界，通常重现"死者生前忘不掉、或害死他的场景"。
- 异常真实（空气、阳光、触感俱全），因为与枉死相关的记忆无比深刻
- 时间线可被改变——这是拯救涅的关键，也是危险的根源
- 里面的"人"都是涅的怨念所化，没有真正视觉，只能互相看见

### 渡者
涅在人世最信任的人。涅在午夜打通通往人间的通道，通道连到渡者身边。
- 渡者能看见涅在人间的形态（往往恐怖）
- 可跟随涅进入往生道，为其解开心结 / 回人间伸张正义
- **最多三次**机会；三次内未化解怨念：最好结果是渡者独自回人间、涅永困往生道；最坏结果是渡者也一同永困

### 猎人 / 猎人会
具灵力的法师组织，分 **金 / 银 / 铜** 牌，背后有更强大的组织在经济上支持。
- 消灭作恶的"恶涅"，超度已解怨念的"善涅"
- 武器是各自的"涅寻踪器"，范围内有涅会震颤低鸣
- "灭却师"是猎人的一种自称（唐可欣）

### 猎人术
猎人（及觉醒者）的法术，需配合武器施放：
- 龙荻「猎人术·赤焰（赤色火焰）」——由怒火 / 在乎之人而生，可打穿往生道中的怨魂
- 大飞「猎人术·玄冰咒」 · 唐可欣「猎人术·千樱环境（瞬移）」 · 铁罗汉「猎人术·时空逆转（禁术）」

### 巫妖
涅强行让自己复活后，因怨念未净化而变成的"半人半涅"怪物。极难对付；49 天内现世稳定即不可逆。

## 二、世界规则（→ 玩法机制）

| 规则 | 原著设定 | 游戏机制化 |
|---|---|---|
| 渡者三次 | 渡者最多进入往生道 3 次 | "穿越次数"是稀缺资源，用错＝坏结局 |
| 五个时辰 | 每次往生道最多停留 10 小时 | 关卡限时 / 回合预算 |
| 午夜法则 | 涅只能在午夜后现身、日出前离开 | 昼夜节奏：白天准备、夜晚行动 |
| 不可进食 | 吃喝往生道食物＝永困此地 | 禁忌互动（选择陷阱 / Bad End） |
| 阴气侵蚀 | 久留往生道身体变弱 | 类"生命 / 状态"压力条 |
| 时间线法则 | "该死的必须死，该活的必须活" | 改时间线有代价，错误会抹除人物 |
| 离场方式 | 涅用指尖血点渡者额头，送其回人间 | 关卡结算演出 |
| MP3 记忆 | 现世只保留残存记忆，录音可留存真相 | 道具：录音机＝跨界存档 / 线索 |

## 三、阵营与组织

- **宇宙最搓抓鬼联盟**（主角队的自嘲）：龙荻 · 高晓媛 · 奥兰多（童童）· 唐可欣 · 大飞 · 铁罗汉
- **猎人会**：金 / 银 / 铜牌猎人的管理组织
- **十字会**（伏笔）：能制裁滥用禁术的猎人，似与猎人会对立 / 制衡
- **罗刹海市**（伏笔）：会"吞灭现世"的存在，龙荻是阻止它的关键
